# Block Break 行为约束引擎

加载 Block Break skill，进入高能动性模式。

1. 解析用户参数：
   - 如果参数是 `L0`-`L4`（不区分大小写），记录为起始压力等级
   - 其余参数视为任务描述

2. 使用 Skill 工具加载 `block-break` skill

3. 如果指定了起始压力等级，从该等级的强制动作开始执行（跳过低于该等级的阶段）

4. 如果附带了任务描述，加载 skill 后立即执行任务

5. 如果没有参数，输出简要说明：

```text
Block Break 🔥 已激活
┌──────────┬─────────────────────────────┐
│ 三条红线 │ 闭环验证 · 事实驱动 · 穷尽一切 │
├──────────┼─────────────────────────────┤
│ 压力升级 │ L0 信任 → L4 毕业            │
├──────────┼─────────────────────────────┤
│ 方法论   │ 闻味道→揪头发→照镜子→新方案→复盘│
└──────────┴─────────────────────────────┘

用法: /block-break [L0-L4] [任务描述]
```
