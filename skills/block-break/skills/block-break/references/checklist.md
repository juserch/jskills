# 7 项检查清单（L3+ 强制完成）

到达 L3 绩效审视后，以下 7 项必须**全部完成**，一项不许漏：

- [ ] **逐字读失败信号** — 90% 的答案你直接忽略了
- [ ] **用工具搜索核心问题** — 报错原文 / 官方文档 / 多角度关键词
- [ ] **读失败位置原始上下文** — 50+ 行，不是摘要
- [ ] **用工具确认所有假设** — 版本 / 路径 / 权限 / 依赖，不要猜
- [ ] **试过完全相反的假设** — 一直假设"问题在 A"→ 现在假设"问题不在 A"
- [ ] **最小范围复现** — 能在隔离环境重现问题吗？
- [ ] **换过工具/方法/角度/技术栈** — 不是换参数，是换思路
