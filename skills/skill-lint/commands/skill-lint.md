# Skill Lint — Claude Code Skill 校验

校验 Claude Code plugin 项目中 skill 的结构完整性和语义质量。

1. 解析用户参数：
   - 如果有路径参数，作为目标 plugin 根路径
   - 无参数则使用当前工作目录

2. 使用 Skill 工具加载 `skill-lint` skill

3. 按 skill 定义的工作流执行：先运行 bash 结构检查脚本，再执行 AI 语义检查

4. 输出合并结果表格

5. 如果没有参数，输出简要说明：

```text
Skill Lint 已激活
┌──────────┬───────────────────────────────────┐
│ 结构检查 │ frontmatter · 文件 · 引用 · 条目   │
├──────────┼───────────────────────────────────┤
│ 语义检查 │ 描述质量 · 一致性 · 路由 · 覆盖度   │
└──────────┴───────────────────────────────────┘

用法: /skill-lint [path]
```
